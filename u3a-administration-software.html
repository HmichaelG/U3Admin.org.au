U3A Administration Website

The Administration Website provides access to authorised users only. An authorised user is one that has successfully authenticated themselves by logging into the site and has been assigned one or more administrative roles by the system administrator.

The administration Website contains all the procedures necessary to administer and mange your membership register, course and class definition, student enrolment and class attendance.

We have reports, and we have analysis. We present that analysis on beautiful dashboards ready for presentation to your committee.
